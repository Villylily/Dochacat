# Dochacat:3
### ⚠️ Notice: Where did the old releases go?
If you are looking for versions of Dochacat:3 older than V11.5, they have been permanently removed. 

**Why?** We recently updated our licensing. Dochacat:3 was originally launched under the MIT license, which technically allowed anyone to take the code, rebrand it, and sell it for profit. We didn't want that. This project is built for friends, communities, and hobbyists—not for corporations to make a quick buck.

To enforce this, we switched to the **PolyForm Noncommercial License**. To prevent any confusion or loopholes with the old rules, all previous MIT-licensed releases have been completely scrubbed from the repository. 

Moving forward, V11.5+ is the only official, supported timeline for Dochacat:3.
So please wait a bit
